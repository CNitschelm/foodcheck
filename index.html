Account

GitHub sync (cross-device)

Token is stored only in this browser. Set it up once per device. Each device pulls/merges/pushes data/items.json + photos in your repo automatically.

AI auto-fill (optional)

With a key set, uploading a photo identifies the product and fills name, brand, category and nutrition (read from the label when visible, otherwise typical values for that product) — you just pick Good/Bad and save. Provider auto-detected from the key. Free key: aistudio.google.com (Google Gemini, no card needed). Paid: console.anthropic.com. Key stays in this browser only.

One-password login

Encrypts this device's keys (sync token + AI key) with a password you choose. Once the bundle is published as config.enc in the app repo, any new device just asks for that password — no key pasting. Keys are never public in plain text (AES-256, PBKDF2).

Backup file

Everything already backs up online automatically — this is just an extra manual copy (photos included).
